<h1>Notary site</h1>

<h2>Используемые технологии:</h2>
 - Gulp 4.0
 - Pug(Jade)
 - Sass(Scss)
 - JS+jQuery

<h2>Start:</h2>
<ol>
  <li>install node.js</li>
  <li>clone this repo</li>
  <li>cd path/to/</li>
  <li>npm i gulpjs/gulp-cli -g  // Install the latest Gulp CLI tools globally</li>
  <li>npm i</li>
  <li>run "gulp" command to start</li>
</ol>
